# Recommender

Recommender that uses a mixture of data from my watch history, my ratings, what is in my Plex library, what is on Anilist to recommend me stuff to watch.

## Data Sources

 - My Plex library
 - My Anilist library
 - My Letterboxd ratings
 - My Traktv watch history

## Future Data Sources

 - Goodreads
 - My Kindle Library
 - My Spotify Library
 - My Kavita Library
